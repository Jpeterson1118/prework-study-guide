HTML

  • The <head> element contains information about the webage.
  • The <body> element represents the visible content shown the user.
  • The <style> element defines the CSS styles associated with a website.
  • The <base> element defines the base URL for a webpage.
  • The <link> element connects an external resource to the HTML document.
  • The <meta> element defines metadata such as the character set, description, keywords, author, and viewport.
  • The charset element sets our character encoding.
  • The http-equiv element sets an HTTP header for content.
  • The viewport element sets the default view for our webpage.
  • <body> semantic HTML containing user viewable content
  • <header> semantic HTML denoting the header section
  • header sits at the top ofthe page and usually contains navigation and/or introductory content
  • heading tags denote page/section headings
  • smaller and less important headings in ascending order 1-6
  • <h1>, < h2>, <h3>, <h4>, <h5>, and <h6>

CSS

  • Three methods of styling web page
  • inline: styles set within content tags
  • highest specificity, overwrites any other styles
  • will we learn about specificty in the prework? Tune in next time same CSS time, same CSS network
  • internal stylesheet: contained within <style> tags in head
  • higher specificity than external CSS, lower than inline
  • reduces potental conflicts with other code
  • I would honestly just link an external stylesheet in most cases, though
  • external stylesheet: seperate document, linked with <link> tags in head
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is a version control sytem for programmers
  • GitHub allows collaberative coding and version control
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • -m adds a message to our commits
  • git add: stages changes
  • -A indicates all changes will be staged
